# Location

Fields relating to the location at which a document was written.

| Field Name                                              | Description                                                                                                                                                                                                                                                                                              | Filter Type                 | Field Type        | Owner    |
|---------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------|-------------------|----------|
| metaData.source.location.countryCode<br><br>**country** | This is the *lowercased* 2 letter [ISO 3166-1 Alpha-2](https://en.wikipedia.org/wiki/ISO_3166-1#Current_codes) code for the country, for example "us" or "se". Defaults to "zz" if the country is unknown.                                                                                               | Exact match                 | String            | Gecko |
| metaData.source.location.voiv<br><br>*region*           | Place name given by the Meltwater news crawler.                                                                                                                                                                                                                                                          | Exact match                 | String            | Gecko |
| metaData.source.location.geonames.continent             | [Geonames.org](https://www.geonames.org) continent ID, e.g. 6255149 for North America.                                                                                                                                                                                                                         | Exact match                 | Integer           | Gecko |
| metaData.source.location.geonames.country               | [Geonames.org](https://www.geonames.org) country ID, e.g. United States or United Kingdom.                                                                                                                                                                                                               | Exact match                 | Integer           | Gecko |
| metaData.source.location.geonames.adm1                  | [Geonames.org](https://www.geonames.org) adm1 ID, e.g. California or England.                                                                                                                                                                                                                            | Exact match                 | Integer           | Gecko |
| metaData.source.location.geonames.adm2                  | [Geonames.org](https://www.geonames.org) adm2 ID, e.g. San Francisco County or Greater London Area.                                                                                                                                                                                                      | Exact match                 | Integer           | Gecko |
| metaData.source.location.geonames.place                 | [Geonames.org](https://www.geonames.org) place ID, e.g. San Francisco or London.                                                                                                                                                                                                                         | Exact match                 | Integer           | Gecko |
| metaData.source.location.geonames.regions               | All [Geonames.org](https://www.geonames.org) region IDs this location is a part of, such as geographic areas of a country (*US Northeast*), sub-continental regions (*Southeast Asia*), as well as continental regions (*Americas*). A location may be a part of multiple partially overlapping regions. | Exact match                 | Array of integers | Gecko |
| metaData.source.location.coordinates.lat                | Latitude                                                                                                                                                                                                                                                                                                 | *Not supported in queries.* | Float             | Gecko |
| metaData.source.location.coordinates.lon                | Longitude                                                                                                                                                                                                                                                                                                | *Not supported in queries.* | Float             | Gecko |
